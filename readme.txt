剑网三自动回复机器人：
*不保证不会被封号
*请遵守GPL v3协议
*不兼容python2，win10 home 14393+py 3.6 通过

安装：
1. win8以上请手动安装winio/winio64.sys，并启动测试模式
(管理员cmd: bcdedit /set testsigning on)

2. 将main.py中的路径改为茗伊插件【你需要的角色】聊天记录保存的路径
   修改settings.py相关设置
3. 快捷键设置中取消风云榜快捷键CTRL+V的设置，并按照VIP1.py
4. 茗伊插件聊天记录设置为实时写入数据库
5. 安装依赖库 rabird.winio, pyperclip
6. 使用管理员权限运行main.py
7. 登录2中的角色，保持剑三运行前台
8. 设置收信邮箱将自己列为白名单
